# Description
- 엔진 : UE 4.25.4 
- 협업 툴 : Perforce 
- 개발 언어 : C++(캐릭터,몬스터 로직), BluePrint(맵 기믹,UI) 
- 개발 기간:(21.10.15~21.12.20) 
- 게임 컨셉: 
    - 비행기를 타고 어딘가로 팔려가던 동물원 펭귄이 어떠한 사고로 사막에 떨어짐
    - 우연히도 펭귄만 생존
    - Hunger수치를 관리하며 사막에서 생존하면서 게임을 진행
    - 사막 어딘가에 떨어진 비행기 부품 5개를 조합해서 탈출하는 것이 최종 목표
